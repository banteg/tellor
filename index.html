Tellor Python Tools¶

A collection of Python tools to interact with Tellor decentralized oracle.

Quickstart¶

If you are running a local node, you can simply use tellor.auto to get a wrapper instance configured for the network your node is on. Both mainnet and rinkeby are supported.

from tellor.auto import tellor

If your node uses a non-standard configuration or you use Infura, update the WEB3_HTTP_PROVIDER_URI environment variable.

>>> tellor.network
<Network.MAINNET: 1>
>>> tellor.address
'0x0Ba45A8b5d5575935B8158a88C631E9F9C95a2e5'

# Wei values transform to a readable form when printed
>>> tellor.total_supply
520910000000000000000000
>>> print(tellor.total_supply)
520910 trb

# Disputes and oracle values are fully supported
